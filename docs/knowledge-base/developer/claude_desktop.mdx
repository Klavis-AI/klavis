## Quick Setup Guide

<Steps>
  <Step title="Access Klavis MCP Servers">
    Navigate to the [Klavis home page](https://www.klavis.ai/home) and click **"MCP Server"** in the left sidebar, pick MCP Server you like.
    
    <img
      src="/images/knowledge-base/claude_desktop/step1.png"
      alt="Klavis home page"
    />    
    Click the **"Create"** button next to your chosen server.
  </Step>

  <Step title="Complete Authentication">
    <img
      src="/images/knowledge-base/claude_desktop/step2.png"
      alt="Klavis MCP Servers page"
    />
    
    <img
      src="/images/knowledge-base/claude_desktop/step3.png"
      alt="OAuth authentication flow"
    />
    
    <img
      src="/images/knowledge-base/claude_desktop/step4.png"
      alt="Server URL generated"
    />
    
    <Accordion title="🔐 OAuth Services (Most Common)">
      - Automatically redirected to OAuth authorization
      - Sign in to your account (GitHub, Google, Slack, etc.)
      - Grant necessary permissions
      - Redirected back to Klavis automatically
    </Accordion>
    
    <Accordion title="🔑 API Key Services">
      - Prompted to enter an API key
      - Follow service-specific instructions to generate key
      - Paste key in the provided field
      - Click **"Save"** to continue
    </Accordion>
  </Step>



  <Step title="Configure Claude Desktop">
    Copy Your Server URL, open Claude Desktop settings:
    - **macOS**: `Cmd + ,` or **Claude Desktop > Settings > Developer**
    - **Windows/Linux**: `Ctrl + ,` or **Claude Desktop > Settings > Developer**
    
    Paste Your Server URL to **claude_desktop_config.json** file like the screenshot below


    <img
      src="/images/knowledge-base/claude_desktop/step5.png"
      alt="Claude Desktop Settings"
    />
    <img
      src="/images/knowledge-base/claude_desktop/step6.png"
      alt="claude_desktop_config.json file in Claude Desktop"
    />

  </Step>

  <Step title="Verify Tools Loading">
     
    <img
      src="/images/knowledge-base/claude_desktop/step7.png"
      alt="MCP Configuration in Claude Desktop"
    />

    <img
      src="/images/knowledge-base/claude_desktop/step8.png"
      alt="MCP Configuration and Tool List in Claude Desktop"
    />
    
    <Note>
      📋 **Reopen Claude Desktop Settings** to apply the new configuration
    </Note>
    
  </Step>

  <Step title="Start Using in Claude Desktop Chat">
    
    <img
      src="/images/knowledge-base/claude_desktop/step9.png"
      alt="Using MCP Server in Claude Desktop Chat"
    />
    
    Open Claude Desktop Chat (`Cmd/Ctrl + L`) and start using natural language:
    
    <CodeGroup>
    ```text GitHub
    "Create a new issue titled 'Add dark mode' with priority label"
    ```
    
    ```text Slack
    "Send a message to #general: 'Standup meeting in 5 minutes!'"
    ```
    
    ```text Gmail
    "Send email to john@company.com about project update"
    ```
    
    ```text Notion
    "Create a new page called 'Meeting Notes' with today's date"
    ```
    </CodeGroup>
    
    <Tip>
      🎯 Claude Desktop Chat automatically detects when to use your MCP server tools based on context - no need to specify tool names!
    </Tip>
  </Step>

  <Step title="Happy Building! 🎉">
    You're all set! Your MCP server is now integrated with Claude Desktop.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="🔧 Tools Not Appearing">
    - Double-check your Server URL for typos
    - Ensure stable internet connection
    - Verify authentication in Klavis dashboard
    - Check Claude Desktop logs for error messages
    - Try completely restarting Claude Desktop
  </Accordion>
  
  <Accordion title="🔐 Authentication Issues">
    - Re-authenticate in the Klavis dashboard
    - Check if your OAuth tokens have expired
    - Verify API key permissions (for API key services)
    - Ensure you've granted all necessary permissions
  </Accordion>
  
  <Accordion title="⚡ Performance Issues">
    - Limit the number of active MCP servers
    - Check your internet connection speed
    - Restart Claude Desktop periodically
    - Contact support if issues persist
  </Accordion>
</AccordionGroup>
