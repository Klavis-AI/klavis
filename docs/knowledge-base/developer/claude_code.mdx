## Quick Setup Guide

<Steps>
  <Step title="Access Klavis MCP Servers">
    Navigate to the [Klavis home page](https://www.klavis.ai/home) and click **"MCP Server"** in the left sidebar, pick MCP Server you like.
    
    <img
      src="/images/knowledge-base/claude_code/step1.png"
      alt="Klavis home page"
    />    
    Click the **"Create"** button next to your chosen server.
  </Step>

  <Step title="Complete Authentication">
    <img
      src="/images/knowledge-base/claude_code/step2.png"
      alt="Klavis MCP Servers page"
    />
    
    <img
      src="/images/knowledge-base/claude_code/step3.png"
      alt="OAuth authentication flow"
    />
    
    <img
      src="/images/knowledge-base/claude_code/step4.png"
      alt="Server URL generated"
    />
    
    <Accordion title="🔐 OAuth Services (Most Common)">
      - Automatically redirected to OAuth authorization
      - Sign in to your account (GitHub, Google, Slack, etc.)
      - Grant necessary permissions
      - Redirected back to Klavis automatically
    </Accordion>
    
    <Accordion title="🔑 API Key Services">
      - Prompted to enter an API key
      - Follow service-specific instructions to generate key
      - Paste key in the provided field
      - Click **"Save"** to continue
    </Accordion>
  </Step>



  <Step title="Configure Claude Code">
    Copy Your Server URL, open terminal and run the below command as shown in the screenshot below:
    ```bash
    claude mcp add serverName npx mcp-remote instance_url
     ```
    <img
      src="/images/knowledge-base/claude_code/step5.png"
      alt="Claude Code command to add Klavis MCP Server"
    />

  </Step>

  <Step title="Verify Tools Loading">

    To verify MCP server is configured properly, open Claude Code and run `/mcp` command.

    <img
      src="/images/knowledge-base/claude_code/step6.png"
      alt="MCP Configuration in Claude Code"
    />

    To view available tools in the MCP server, navigate to specific server and click `View Tools`.
     
    <img
      src="/images/knowledge-base/claude_code/step7.png"
      alt="View Tools in Claude Code"
    />
    
    <Note>
      📋 **Restart Claude Code** in terminal to apply the new configuration
    </Note>
    
  </Step>

  <Step title="Start Using in Claude Code Chat">
    <img
      src="/images/knowledge-base/claude_code/step8.png"
      alt="Using MCP Server in Claude Code Chat"
    />
    
    Open Claude Code Chat and start using natural language:
    
    <CodeGroup>
    ```text GitHub
    "Create a new issue titled 'Add dark mode' with priority label"
    ```
    
    ```text Slack
    "Send a message to #general: 'Standup meeting in 5 minutes!'"
    ```
    
    ```text Gmail
    "Send email to john@company.com about project update"
    ```
    
    ```text Notion
    "Create a new page called 'Meeting Notes' with today's date"
    ```
    </CodeGroup>
    
    <Tip>
      🎯 Claude Code Chat automatically detects when to use your MCP server tools based on context - no need to specify tool names!
    </Tip>
  </Step>


  <Step title="Other Commands related to MCPs">
    - Remove MCP Server: `claude mcp remove serverName`
    - List all MCP Servers: `claude mcp list`
    - Details of specific MCP Server: `claude mcp get serverName`
  </Step>


  <Step title="Happy Building! 🎉">
    You're all set! Your MCP server is now integrated with Claude Code.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="🔧 Tools Not Appearing">
    - Double-check your Server URL for typos
    - Ensure stable internet connection
    - Verify authentication in Klavis dashboard
    - Try completely restarting Claude Code
  </Accordion>
  
  <Accordion title="🔐 Authentication Issues">
    - Re-authenticate in the Klavis dashboard
    - Check if your OAuth tokens have expired
    - Verify API key permissions (for API key services)
    - Ensure you've granted all necessary permissions
  </Accordion>
  
  <Accordion title="⚡ Performance Issues">
    - Limit the number of active MCP servers
    - Check your internet connection speed
    - Restart Claude Code periodically
    - Contact support if issues persist
  </Accordion>
</AccordionGroup>
