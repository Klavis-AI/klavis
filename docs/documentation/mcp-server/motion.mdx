## Getting started

Connect to Motion to manage tasks, projects, workspaces, and automate scheduling workflows through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a server instance">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create a Motion MCP server instance
          motion_server = klavis_client.mcp_server.create_server_instance(
              server_name=McpServerName.MOTION,
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: 'YOUR_API_KEY' });
          
          // Create a Motion MCP server instance
          const motionServer = await klavis.mcpServer.createServerInstance({
              serverName: Klavis.McpServerName.Motion,
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "Motion",
              "userId": "user123"
            }'
          ```
        </CodeGroup>
        
        <Info>
          **Response Information**: The API returns:
          - `serverUrl`: The URL for connecting your MCP client to Motion
          - `instanceId`: Unique identifier for your server instance
        </Info>
      </Step>
      
      <Step title="Authenticate with API Key">
        <CodeGroup>
          ```python Python
          # Set your Motion API key for the instance
          auth_response = klavis_client.mcp_server.set_instance_auth(
              instance_id=motion_server.instance_id,
              auth_data={
                  "token": "YOUR_MOTION_API_KEY"
              }
          )
          ```
          
          ```typescript TypeScript
          // Set your Motion API key for the instance
          const authResponse = await klavis.mcpServer.setInstanceAuth({
              instanceId: motionServer.instanceId,
              authData: {
                  token: "YOUR_MOTION_API_KEY"
              }
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/set-auth" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "instanceId": "YOUR_INSTANCE_ID",
              "authData": {
                "token": "YOUR_MOTION_API_KEY"
              }
            }'
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your Motion MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>

        ```json
          "mcpServers": {
            "motion": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>

## Available Tools

<Accordion title="Tools Information">
| Tool Name | Description |
|-----------|-------------|
| motion_get_workspaces | Get all workspaces in the Motion account |
| motion_get_users | Get all users, optionally filtered by workspace |
| motion_get_my_user | Get current user information |
| motion_get_tasks | Get tasks, optionally filtered by workspace |
| motion_get_task | Get a specific task by its ID |
| motion_create_task | Create a new task in Motion |
| motion_update_task | Update an existing task in Motion |
| motion_delete_task | Delete a task from Motion |
| motion_search_tasks | Search for tasks by name or description |
| motion_get_projects | Get projects, optionally filtered by workspace |
| motion_get_project | Get a specific project by its ID |
| motion_create_project | Create a new project in Motion |
| motion_get_comments | Get comments for a specific task |
| motion_create_comment | Create a comment on a task |
</Accordion>

<Note>For more details about tool input schema, use the [get_tools](https://docs.klavis.ai/api-reference/mcp-server/get-tools) API.</Note>
