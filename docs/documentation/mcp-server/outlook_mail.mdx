## Getting started

Connect to Outlook Mail to read, send, organize emails, and automate email workflows through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a server instance">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create an Outlook Mail MCP server instance
          outlook_server = klavis_client.mcp_server.create_server_instance(
              server_name=McpServerName.OUTLOOK_MAIL,
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: 'YOUR_API_KEY' });
          
          // Create an Outlook Mail MCP server instance
          const outlookServer = await klavis.mcpServer.createServerInstance({
              serverName: Klavis.McpServerName.OutlookMail,
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "Outlook Mail",
              "userId": "user123"
            }'
          ```
        </CodeGroup>
        
        <Info>
          **Response Information**: The API returns:
          - `serverUrl`: The URL for connecting your MCP client to Outlook Mail
          - `instanceId`: Unique identifier for your server instance
          - `oauthUrl`: OAuth authorization URL for Outlook Mail authentication
        </Info>
      </Step>
      
      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser
          
          # Open OAuth authorization page
          webbrowser.open(outlook_server.oauth_url)
          ```
          
          ```typescript TypeScript
          // Redirect user to OAuth authorization
          window.location.href = outlookServer.oauthUrl;
          ```
          
          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.klavis.ai/oauth/outlook/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your Outlook Mail MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      
      <Step title="(optional) Connect to your AI application">
        <Tabs>
          <Tab title="LangChain">
            ```python
            import asyncio
            from langchain_openai import ChatOpenAI
            from langchain_mcp_adapters.client import MultiServerMCPClient
            from langgraph.prebuilt import create_react_agent
            
            async def main():
                # Create MCP client
                mcp_client = MultiServerMCPClient({
                    "outlook_mail": {
                        "transport": "streamable_http",
                        "url": outlook_server.server_url
                    }
                })
                
                # Get tools and create agent
                tools = await mcp_client.get_tools()
                llm = ChatOpenAI(model="gpt-4o-mini")
                
                agent = create_react_agent(
                    model=llm,
                    tools=tools,
                    prompt="You are an email assistant that helps manage Outlook emails, organize messages, and automate email workflows."
                )
                
                # Use the agent
                result = await agent.ainvoke({
                    "messages": [{"role": "user", "content": "Check my inbox and summarize the most important emails"}]
                })
                
                print(result["messages"][-1].content)
            
            asyncio.run(main())
            ```
          </Tab>
        </Tabs>
      </Step>

## Available Tools

<Accordion title="Tools Information">
| Tool Name | Description |
|-----------|-------------|
| outlook_send_email | Send an email through Outlook with support for HTML content, attachments, and priority settings |
| outlook_get_messages | Retrieve messages from inbox, sent items, or specific folders with filtering options |
| outlook_get_message_by_id | Get a specific email message by its unique ID |
| outlook_reply_to_message | Reply to an existing email message |
| outlook_forward_message | Forward an email message to other recipients |
| outlook_delete_message | Delete an email message (move to deleted items) |
| outlook_move_message | Move an email message to a different folder |
| outlook_mark_as_read | Mark an email message as read |
| outlook_mark_as_unread | Mark an email message as unread |
| outlook_get_folders | List all mail folders in the account |
| outlook_create_folder | Create a new mail folder |
| outlook_search_messages | Search for messages using various criteria including keywords, sender, and date ranges |
</Accordion>

<Note>For more details about tool input schema, use the [list_tool](https://docs.klavis.ai/api-reference/mcp-server/list-tools) API.</Note>
