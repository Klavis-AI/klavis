## Getting started

Connect to Cal.com to manage schedules, create appointments, and automate calendar operations through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a server instance">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create a Cal.com MCP server instance
          cal_com_server = klavis_client.mcp_server.create_server_instance(
              server_name=McpServerName.CAL_COM,
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: 'YOUR_API_KEY' });
          
          // Create a Cal.com MCP server instance
          const calComServer = await klavis.mcpServer.createServerInstance({
              serverName: Klavis.McpServerName.CalCom,
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "CalCom",
              "userId": "user123"
            }'
          ```
        </CodeGroup>
        
        <Info>
          **Response Information**: The API returns:
          - `serverUrl`: The URL for connecting your MCP client to Cal.com
          - `instanceId`: Unique identifier for your server instance
          - `oauthUrl`: OAuth authorization URL for Cal.com authentication
        </Info>
      </Step>
      
      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser
          
          # Open OAuth authorization page
          webbrowser.open(cal_com_server.oauth_url)
          ```
          
          ```typescript TypeScript
          // Redirect user to OAuth authorization
          window.location.href = calComServer.oauthUrl;
          ```
          
          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.klavis.ai/oauth/cal_com/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your Cal.com MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      
      <Step title="(optional) Connect to your AI application">
        <Tabs>
          <Tab title="LangChain">
            ```python
            import asyncio
            from langchain_openai import ChatOpenAI
            from langchain_mcp_adapters.client import MultiServerMCPClient
            from langgraph.prebuilt import create_react_agent
            
            async def main():
                # Create MCP client
                mcp_client = MultiServerMCPClient({
                    "cal_com": {
                        "transport": "streamable_http",
                        "url": cal_com_server.server_url
                    }
                })
                
                # Get tools and create agent
                tools = await mcp_client.get_tools()
                llm = ChatOpenAI(model="gpt-4o-mini")
                
                agent = create_react_agent(
                    model=llm,
                    tools=tools,
                    prompt="You are a scheduling assistant that helps manage Cal.com calendars."
                )
                
                # Use the agent
                result = await agent.ainvoke({
                    "messages": [{"role": "user", "content": "Show me all my schedules"}]
                })
                
                print(result["messages"][-1].content)
            
            asyncio.run(main())
            ```
          </Tab>
        </Tabs>
      </Step>
    </Steps>
  </Tab>

    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/klavis-ai/klavis
        cd klavis/mcp_servers/cal_com
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/klavis-ai/cal-com-mcp-server:latest
        
        # Run with OAuth support (requires Klavis API key)
        docker run -p 5000:5000 \
          -e KLAVIS_API_KEY=$KLAVIS_API_KEY \
          ghcr.io/klavis-ai/cal-com-mcp-server:latest
        
        # Or run with manual API key
        docker run -p 5000:5000 \
          -e AUTH_DATA='{"access_token":"your_cal_com_token"}' \
          ghcr.io/klavis-ai/cal-com-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "cal_com": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>

## Available Tools

<Accordion title="Tools Information">
| Tool Name | Description |
|-----------|-------------|
| cal_get_all_schedules | Retrieve all schedules from Cal.com |
| cal_create_a_schedule | Create a new schedule in Cal.com |
| cal_update_a_schedule | Update an existing schedule in Cal.com |
| cal_delete_a_schedule | Delete a schedule by ID in Cal.com |
| cal_get_default_schedule | Get the default schedule configured in Cal.com |
| cal_get_schedule | Fetch a specific schedule by ID from Cal.com |
</Accordion>

<Note>For more details about tool input schema, use the [list_tool](https://docs.klavis.ai/api-reference/mcp-server/list-tools) API.</Note>
