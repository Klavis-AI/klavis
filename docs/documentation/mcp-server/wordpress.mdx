## Getting started

Connect to WordPress.com to manage posts, pages, sites, and automate content publishing workflows through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a server instance">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create a WordPress MCP server instance
          wordpress_server = klavis_client.mcp_server.create_server_instance(
              server_name=McpServerName.WORDPRESS,
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: 'YOUR_API_KEY' });
          
          // Create a WordPress MCP server instance
          const wordpressServer = await klavis.mcpServer.createServerInstance({
              serverName: Klavis.McpServerName.WordPress,
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "WordPress",
              "userId": "user123"
            }'
          ```
        </CodeGroup>
        
        <Info>
          **Response Information**: The API returns:
          - `serverUrl`: The URL for connecting your MCP client to WordPress
          - `instanceId`: Unique identifier for your server instance
          - `oauthUrl`: OAuth authorization URL for WordPress authentication
        </Info>
      </Step>
      
      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser
          
          # Open OAuth authorization page
          webbrowser.open(wordpress_server.oauth_url)
          ```
          
          ```typescript TypeScript
          // Redirect user to OAuth authorization
          window.location.href = wordpressServer.oauthUrl;
          ```
          
          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.klavis.ai/oauth/wordpress/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>
        
        <Check>
          Your WordPress MCP Server is ready! Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      
      <Step title="(optional) Connect to your AI application">
        <Tabs>
          <Tab title="LangChain">
            ```python
            import asyncio
            from langchain_openai import ChatOpenAI
            from langchain_mcp_adapters.client import MultiServerMCPClient
            from langgraph.prebuilt import create_react_agent
            
            async def main():
                # Create MCP client
                mcp_client = MultiServerMCPClient({
                    "wordpress": {
                        "transport": "streamable_http",
                        "url": wordpress_server.server_url
                    }
                })
                
                # Get tools and create agent
                tools = await mcp_client.get_tools()
                llm = ChatOpenAI(model="gpt-4o-mini")
                
                agent = create_react_agent(
                    model=llm,
                    tools=tools,
                    prompt="You are a content management assistant that helps create, edit, and manage WordPress posts and pages."
                )
                
                # Use the agent
                result = await agent.ainvoke({
                    "messages": [{"role": "user", "content": "Create a new blog post about AI trends in 2024"}]
                })
                
                print(result["messages"][-1].content)
            
            asyncio.run(main())
            ```
          </Tab>
        </Tabs>
      </Step>
    </Steps>
  </Tab>

        ```bash
        git clone https://github.com/klavis-ai/klavis
        cd klavis/mcp_servers/wordpress
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/klavis-ai/wordpress-mcp-server:latest

          -e KLAVIS_API_KEY=$KLAVIS_API_KEY \
          ghcr.io/klavis-ai/wordpress-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "wordpress": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>

## Available Tools

<Accordion title="Tools Information">
| Tool Name | Description |
|-----------|-------------|
| wordpress_create_post | Create a new WordPress post |
| wordpress_get_posts | Get a list of WordPress posts |
| wordpress_update_post | Update an existing WordPress post |
| wordpress_get_top_posts | Get top WordPress posts for a site |
| wordpress_get_site_info | Get information about a WordPress site |
| wordpress_get_site_stats | Get statistics for a WordPress site |
| wordpress_get_user_sites | Get all WordPress sites the authenticated user has access to |
</Accordion>

<Note>For more details about tool input schema, use the [list_tool](https://docs.klavis.ai/api-reference/mcp-server/list-tools) API.</Note>
