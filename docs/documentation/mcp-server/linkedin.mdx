## Getting started

Connect to LinkedIn to manage profiles, create posts, share content, and automate professional networking activities through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a server instance">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create a LinkedIn MCP server instance
          linkedin_server = klavis_client.mcp_server.create_server_instance(
              server_name=McpServerName.LINKEDIN,
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: 'YOUR_API_KEY' });
          
          // Create a LinkedIn MCP server instance
          const linkedinServer = await klavis.mcpServer.createServerInstance({
              serverName: Klavis.McpServerName.LinkedIn,
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "LinkedIn",
              "userId": "user123"
            }'
          ```
        </CodeGroup>
        
        <Info>
          **Response Information**: The API returns:
          - `serverUrl`: The URL for connecting your MCP client to LinkedIn
          - `instanceId`: Unique identifier for your server instance
          - `oauthUrl`: OAuth authorization URL for LinkedIn authentication
        </Info>
      </Step>
      
      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser
          
          # Open OAuth authorization page
          webbrowser.open(linkedin_server.oauth_url)
          ```
          
          ```typescript TypeScript
          // Redirect user to OAuth authorization
          window.location.href = linkedinServer.oauthUrl;
          ```
          
          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.klavis.ai/oauth/linkedin/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>
        
        <Tip>
          You can specify custom scope and redirect_url parameters in the OAuth URL. Check the API reference for more details.
        </Tip>
        
        <Check>
          Your LinkedIn MCP Server is ready! Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      
      <Step title="(optional) Connect to your AI application">
        <Tabs>
          <Tab title="LangChain">
            ```python
            import asyncio
            from langchain_openai import ChatOpenAI
            from langchain_mcp_adapters.client import MultiServerMCPClient
            from langgraph.prebuilt import create_react_agent
            
            async def main():
                # Create MCP client
                mcp_client = MultiServerMCPClient({
                    "linkedin": {
                        "transport": "streamable_http",
                        "url": linkedin_server.server_url
                    }
                })
                
                # Get tools and create agent
                tools = await mcp_client.get_tools()
                llm = ChatOpenAI(model="gpt-4o-mini")
                
                agent = create_react_agent(
                    model=llm,
                    tools=tools,
                    prompt="You are a professional networking assistant that helps manage LinkedIn profiles, create posts, and engage with professional content."
                )
                
                # Use the agent
                result = await agent.ainvoke({
                    "messages": [{"role": "user", "content": "Create a LinkedIn post about the future of AI in business"}]
                })
                
                print(result["messages"][-1].content)
            
            asyncio.run(main())
            ```
          </Tab>
        </Tabs>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/klavis-ai/klavis
        cd klavis/mcp_servers/linkedin
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/klavis-ai/linkedin-mcp-server:latest
        
        # Run with OAuth support (requires Klavis API key)
        docker run -p 5000:5000 \
          -e KLAVIS_API_KEY=$KLAVIS_API_KEY \
          ghcr.io/klavis-ai/linkedin-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "linkedin": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>

## Available Tools

<Accordion title="Tools Information">
| Tool Name | Description |
|-----------|-------------|
| linkedin_get_profile_info | Retrieve LinkedIn profile information (current user or specified person) |
| linkedin_create_post | Create a text/article post on LinkedIn with customizable visibility |
| linkedin_create_url_share | Create a LinkedIn post that shares a URL with metadata preview |
| linkedin_format_rich_post | Format rich text for LinkedIn posts with various formatting options |
</Accordion>

<Note>For more details about tool input schema, use the [list_tool](https://docs.klavis.ai/api-reference/mcp-server/list-tools) API.</Note>
