## Getting started

Connect to Klavis ReportGen to generate visually appealing JavaScript web reports based on search queries through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a server instance">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create a Klavis ReportGen MCP server instance
          klavis_reportgen_server = klavis_client.mcp_server.create_server_instance(
              server_name=McpServerName.KLAVIS_REPORTGEN,
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: 'YOUR_API_KEY' });
          
          // Create a Klavis ReportGen MCP server instance
          const klavisReportgenServer = await klavis.mcpServer.createServerInstance({
              serverName: Klavis.McpServerName.KlavisReportgen,
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "Klavis ReportGen",
              "userId": "user123"
            }'
          ```
        </CodeGroup>
        
        <Info>
          **Response Information**: The API returns:
          - `serverUrl`: The URL for connecting your MCP client to Klavis ReportGen
          - `instanceId`: Unique identifier for your server instance
        </Info>
      </Step>
      
      <Step title="Configure API Key">
        <CodeGroup>
          ```python Python
          # Set the Klavis ReportGen API key for your instance
          response = klavis_client.mcp_server.set_instance_auth(
              instance_id=klavis_reportgen_server.instance_id,
              auth_data={
                  "api_key": "YOUR_KLAVIS_REPORTGEN_API_KEY"
              }
          )
          ```
          
          ```typescript TypeScript
          // Set the Klavis ReportGen API key for your instance
          const response = await klavis.mcpServer.setInstanceAuth({
              instanceId: klavisReportgenServer.instanceId,
              authData: {
                  api_key: "YOUR_KLAVIS_REPORTGEN_API_KEY"
              }
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/set-auth" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "instanceId": "YOUR_INSTANCE_ID",
              "authData": {
                "api_key": "YOUR_KLAVIS_REPORTGEN_API_KEY"
              }
            }'
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your Klavis ReportGen MCP Server is ready!** You can now use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      

    </Steps>
  </Tab>

      </Step>

  </Tab>
  
  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/klavis-ai/klavis
        cd klavis/mcp_servers/klavis_reportgen
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/klavis-ai/klavis-reportgen-mcp-server:latest
        
        # Run with API key
        docker run -p 5000:5000 \
          -e AUTH_DATA='{"api_key":"your_klavis_reportgen_api_key"}' \
          ghcr.io/klavis-ai/klavis-reportgen-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "klavis-reportgen": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
