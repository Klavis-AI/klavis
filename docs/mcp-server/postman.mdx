## Getting started

Connect to Postman to manage API collections, run tests, monitor APIs, and collaborate on API development through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a Strata MCP Server with Postman">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          # Create a Strata MCP server with Postman
          response = klavis_client.mcp_server.create_strata_server(
              servers=[McpServerName.POSTMAN],
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';
          
          const klavis = new KlavisClient({ apiKey: process.env.KLAVIS_API_KEY! });
          
          // Create a Strata MCP server with Postman
          const response = await klavis.mcpServer.createStrataServer({
              servers: [Klavis.McpServerName.Postman],
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "servers": ["Postman"],
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Tip>
        `userId` specifies whose connected accounts and data you are accessing in Klavis. It should be a unique id for yourself, your team, or your organization.
        </Tip>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>
      
      <Step title="Configure authentication">
        <Info>
          You can get your Postman API key from the [Postman API Keys page](https://go.postman.co/settings/me/api-keys).
        </Info>
        
        <CodeGroup>
          ```python Python
          # Set the Postman API key for your instance
          response = klavis_client.mcp_server.set_instance_auth(
              instance_id=postman_server.instance_id,
              auth_data={
                  "api_key": "YOUR_POSTMAN_API_KEY"
              }
          )
          ```
          
          ```typescript TypeScript
          // Set the Postman API key for your instance
          const response = await klavis.mcpServer.setInstanceAuth({
              instanceId: postmanServer.instanceId,
              authData: {
                  api_key: "YOUR_POSTMAN_API_KEY"
              }
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/instance/set-auth" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "instanceId": "YOUR_INSTANCE_ID",
              "authData": {
                "api_key": "YOUR_POSTMAN_API_KEY"
              }
            }'
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your Postman MCP Server is ready!** You can now use your MCP server URL with any MCP-compatible client to manage API collections and run tests.
        </Check>
      </Step>
      

    </Steps>
  </Tab>
  
  <Tab title="UI">
    <Steps>
      <Step title="Open Dashboard">
        Go to your [Dashboard](https://www.klavis.ai/home/mcp-servers).
      </Step>
      
      <Step title="Choose Postman">
        Select Postman from the list of available integrations.
      </Step>
      
      <Step title="Configure API Key">
        Enter your Postman API key to enable collection management and testing features.
      </Step>
      
      <Step title="Use in your app">
        Copy the MCP endpoint URL and add it to your MCP-supported client (Claude Desktop, Cursor, VS Code, etc.).
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/klavis-ai/klavis
        cd klavis/mcp_servers/postman
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/klavis-ai/postman-mcp-server:latest
        
        # Run with API key support (requires Klavis API key)
        docker run -p 5000:5000 \
          -e KLAVIS_API_KEY=$KLAVIS_API_KEY \
          ghcr.io/klavis-ai/postman-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "postman": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
