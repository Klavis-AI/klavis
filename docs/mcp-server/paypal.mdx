## Getting started

Connect to PayPal to manage payments, process transactions, and handle invoicing through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>

      <Step title="Create a Strata MCP Server with PayPal">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName

          klavis_client = Klavis(api_key="YOUR_API_KEY")

          response = klavis_client.mcp_server.create_strata_server(
              servers=[McpServerName.PAYPAL],
              user_id="user123"
          )
          ```

          ```typescript TypeScript
          import { KlavisClient, Klavis } from 'klavis';

          const klavis = new KlavisClient({ apiKey: process.env.KLAVIS_API_KEY! });

          // Create a Strata MCP server with PayPal
          const response = await klavis.mcpServer.createStrataServer({
              servers: [Klavis.McpServerName.Paypal],
              userId: "user123"
          });
          ```

          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "servers": ["Paypal"],
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Tip>
        `userId` specifies whose connected accounts and data you are accessing in Klavis. It should be a unique id for yourself, your team, or your organization.
        </Tip>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>

      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser

          # Open OAuth authorization page
          webbrowser.open(response.oauth_urls[McpServerName.PAYPAL])
          ```

          ```typescript TypeScript
          import open from 'open';

          // Open OAuth authorization page
          await open(response.oauthUrls[Klavis.McpServerName.Paypal]);
          ```

          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.klavis.ai/oauth/paypal/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>

        <Check>
          **Your PayPal MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>


    </Steps>
  </Tab>

  <Tab title="UI">
    <Steps>
      <Step title="Open Dashboard">
        Go to your [Dashboard](https://www.klavis.ai/home/mcp-servers).
      </Step>

      <Step title="Choose PayPal">
        Select PayPal from the list of available integrations.
      </Step>

      <Step title="Authenticate">
        Complete the OAuth flow to connect your PayPal account.
      </Step>

      <Step title="Use in your app">
        Copy the MCP endpoint URL and add it to your MCP-supported client (Claude Desktop, Cursor, VS Code, etc.).
      </Step>
    </Steps>
  </Tab>
</Tabs>
